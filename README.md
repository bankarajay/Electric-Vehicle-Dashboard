# Electric Vehicle Analysis Dashboard

This repository contains an interactive dashboard that provides insights into electric vehicle (EV) adoption, distribution, and performance metrics across different states, makes, and models.

##  Dashboard Overview

The dashboard includes:
- **Key Metrics**
  - Total Vehicles: 33.67K
  - Average Electric Range: 30.65 miles
- **Vehicle Types**
  - BEV (Battery Electric Vehicles): 117K
  - PHEV (Plug-in Hybrid Electric Vehicles): 34K
- **Trends**
  - Line chart showing total vehicles by model year (2013â€“2022)
- **Geographic Distribution**
  - Map visualization of EVs by state
- **Manufacturers**
  - Bar chart of vehicles by make 
- **Eligibility**
  - Donut chart showing Clean Alternative Fuel Vehicle (CAFV) eligibility

##  Features

- Total vehicles and average electric range
- Breakdown of BEV and PHEV vehicles
- Trends by model year
- Distribution by state
- Vehicles by make and model
- Eligibility status (CAFV)


##  Tech Stack

- **Data Processing:** Python (pandas)
- **Visualization:** Power BI 
- **Deployment:** GitHub Pages / Local environment
